
# Mi Portafolio Personal 🚀

¡Bienvenido a mi portafolio! Este es un proyecto web sencillo y elegante donde comparto quién soy, mis intereses en el mundo digital y mis proyectos actuales.

## 👤 Sobre mí
Mi nombre es **Jonathan Rodriguez**, tengo 18 años y soy un apasionado del desarrollo web y la tecnología. Me motiva aprender constantemente y enfrentar los retos que ofrece el mundo de la programación.

## 🛠️ Tecnologías utilizadas
Para la construcción de este sitio he utilizado:
* **HTML5**: Estructura semántica del contenido.
* **CSS3**: Diseño visual y estilos personalizados.
* **JavaScript**: Interactividad básica y manejo de formularios.

## 📂 Estructura del Proyecto
El sitio se divide en las siguientes secciones:
* **Inicio**: Introducción personal y perfil.
* **Proyectos**: Enlaces a mis trabajos destacados, incluyendo una sección dedicada a mascotas.
* **Contacto**: Formulario funcional para recibir mensajes de visitantes.

## 🚀 Instalación y Uso
Si quieres ver este proyecto de forma local, sigue estos pasos:

1.  **Clona el repositorio**:
    ```bash
    git clone https://github.com/tu-usuario/mi-portafolio.git
    ```
2.  **Abre el archivo principal**:
    Navega a la carpeta y abre `index.html` en tu navegador preferido.
